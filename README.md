# HTTP Client Demo - Application de Démonstration HTTP

Application React pour démontrer et tester les communications HTTP avec JSON Server (Mock API).

## Installation et démarrage

```bash
# 1. Installation des dépendances
npm install
npm install -g json-server

# 2. Démarrer le serveur JSON Server (dans un terminal)
json-server --watch db.json --port 3001

# 3. Démarrer l'application React (dans un autre terminal)
npm start
